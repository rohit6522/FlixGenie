# 🎬 FlixGenie

An AI-powered movie discovery web app that lets users browse trending movies, get personalized recommendations from an AI assistant, and explore detailed movie information — all wrapped in a Netflix-inspired UI.

## ✨ Features

- 🔐 **Authentication** — Sign up / Sign in with Firebase (Email & Password)
- 🤖 **AI Recommendations** — Natural language movie suggestions powered by Groq (Llama 3.3 70B)
- 🎥 **Movie Discovery** — Browse Trending, Popular, and Top Rated movie rows
- 🖼️ **Hero Banner** — Netflix-style featured movie showcase
- 📋 **Movie Details** — Click any poster for full details (plot, cast, ratings, awards)
- 📱 **Responsive Design** — Works seamlessly across devices

## 🛠️ Tech Stack

**Frontend:** React (Vite), Redux Toolkit, Tailwind CSS v4, React Router, Axios, Firebase Auth
**Backend:** Node.js, Express, Groq API (Llama 3.3), OMDb API
